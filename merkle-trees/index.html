Folie auf der ein Hash in dem Baum veraendert wurde und der Pfad eingezeichnet ist, der sich dann veraendert und dass der root hash dann auch anders ist. --> it's easily possible to make sure that even small changes already affect the integrity of the entire tree --> you could also do this by just concatenating everything and then hashing --> this would be extremely inefficient. with merkle trees you just need to rehash the path to the leaf --> O(log(n))? --> that's why merkle trees are very well suited for e.g. filesystems or git

Scenario: I have a transaction T and want to know if it was included in the latest block. What is needed to proof that a leaf is in the node? --> just the path

Scenario: I know the HEAD of the database. Hey database, tell me the entry at position 5871. --> I get returned Entry + Path up to Merkle Root --> I can verify the info is correct by rehashing Entry + Path = Merkle Root' and comparing MerkleRoot' == Merkle Root --> Authenticate a small amount of data in a potentially unbounded large dataset.

Scenario in Blockchains: I know the HEAD of the (longest) chain. Hey full node, was transaction X in the last block? --> I get returned Transaction + Path up to Merkle Root --> I can verify the info is correct by rehashing Transaction + Path = Merkle Root' and comparing MerkleRoot' == Merkle Root

The way as described so far works very well for:
  • Simple Blockchains with key-value pairs of e.g. Account-Balance (as in Bitcoin)
  • Databases
  • Version Control Systems
  • Filesystems

Shortcomings of Merkle Tress so far: They cannot proof anything about the current state:
  • What is the balance of an accout? You can ask a full node, but there is no way to verify it.
  • Who is the owner of some digital asset?
  • What is the status of a smart contract?

  • Was transaction T included in this block?

  • Fetch me all events of a type T emmited by address A in the last two weeks
    (e.g. a funding goal was reached)
  • What is the current balance of an account?
  • Does account with address A exist?
